To build and install  mysql lib from source(Make sure source is installed from maintance tool)

1. Run build.bat (open build.bat and update qt path and mysql path)

2. Copy following lib to project folder

C:\Program Files\MySQL\MySQL Server 8.0\lib\libmysql.dll
C:\Program Files\MySQL\MySQL Server 8.0\bin\libcrypto-1_1-x64.dll
C:\Program Files\MySQL\MySQL Server 8.0\bin\libssl-1_1-x64.dll
